Class Libros

Representa un libro en la base de datos. Este modelo interactúa con la tabla 'libros' y define las propiedades asignables masivamente ($fillable) y las relaciones Eloquent con otros modelos como Autores, Editoriales, Comentarios, Clientes y Detallespedidos.
