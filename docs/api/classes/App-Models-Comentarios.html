Class Comentarios

Representa un comentario (y opcionalmente una puntuación) realizado por un usuario sobre un libro específico. Este modelo interactúa con la tabla 'comentarios' y define las propiedades asignables masivamente ($fillable) y las relaciones inversas ('belongsTo') con los modelos User y Libros.
