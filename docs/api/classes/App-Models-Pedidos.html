Class Pedidos

Representa un pedido realizado por un usuario en la aplicación. Este modelo interactúa con la tabla 'pedidos' y define las propiedades asignables masivamente ($fillable), los tipos de datos ($casts), constantes para los estados del pedido, y las relaciones Eloquent con los modelos User (cliente) y Detallespedidos (ítems del pedido).

public cliente() : BelongsTo

Establece que cada instancia de Pedidos está asociada a una única instancia de User (el cliente). Permite acceder al cliente relacionado a través de la propiedad $pedido->cliente. La clave foránea utilizada es cliente_id.
